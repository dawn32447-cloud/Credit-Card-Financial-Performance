# Credit Card Financial Performance Dashboard

An end-to-end Power BI data analytics project that develops a comprehensive, dynamic credit card weekly dashboard. This solution provides real-time insights into key performance metrics and operational trends, enabling financial stakeholders to monitor and analyze business metrics efficiently.


## 📌 Project Overview
The objective of this project is to track and evaluate credit card performance KPIs. By utilizing automated data handling, detailed metrics analysis, and advanced DAX queries, this project transforms raw transactional and customer data into a highly interactive financial tracking mechanism.


### Key Insights Available:
* Quarterly & Weekly Trends: Track operations down to week-level performance (e.g., Week 53, Q4 updates).
* Revenue Drivers: Deep dive into revenue breakdown based on expenditure type (Bills, Entertainment, Fuel, Grocery, Food, Travel).
* Customer Demographics: Segregated performance metrics based on customer traits (Education, Profession, Gender, and Income levels).
* Risk & Operations Monitoring: Analyze utilization ratios, interest earned, and identify delinquent accounts.

---


## 📊 Dashboard Architecture & Features
The project features two distinct specialized report view layers built into a unified Neural Expressive UI theme:
1. Credit Card Transaction Report: Monitored KPIs such as Total Transaction Amounts, Revenue generated by categories, and Card types (Blue, Silver, Gold, Platinum).
2. Credit Card Customer Report: Focused heavily on demographics, customer acquisition costs (CAC), satisfaction ratings, and risk metrics.

---




## 🛠️ Tech Stack & Methods Used
* Power BI Desktop: Dashboard engineering, reporting, visual design.
* Power Query Editor: Data cleaning, conditional indexing, and transformation.
* DAX Formulas: Statistical modeling, rolling performance counters, and column mapping.
* Data Sources: CSV files (simulated SQL structured database extracts).
